quick sort
快速排序是一种基于分治思想 的高效排序算法。
它的核心思想是“分而治之”，通过一趟排序将待排记录分隔成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，然后分别对这两部分记录继续进行排序，以达到整个序列有序。

核心思想与步骤
选择基准：从数列中挑出一个元素，称为“基准”。(pivot point )

分区操作：重新排列数列，所有比基准值小的元素摆放在基准前面，所有比基准值大的元素摆在基准后面（相同的数可以到任一边）。在这个分区退出之后，该基准就处于数列的中间位置。这个称为分区操作。

递归排序：递归地将小于基准值的子数列和大于基准值的子数列排序。

演示（以第一个元素为基准）
假设我们要排序的数组是：[6, 5, 3, 1, 8, 7, 2, 4]

第一轮：

基准 pivot = 6

分区过程：

从右向左找第一个小于6的数 4，从左向右找第一个大于6的数 8，交换。 -> [6, 5, 3, 1, 4, 7, 2, 8]

继续，从右向左找到 2，从左向右找到 7，交换。 -> [6, 5, 3, 1, 4, 2, 7, 8]

指针相遇，将基准 6 与相遇点的 2 交换。 -> [2, 5, 3, 1, 4, 6, 7, 8]

现在，6 已经在正确的位置上，左边 [2, 5, 3, 1, 4] 都比6小，右边 [7, 8] 都比6大。

递归：

对左子数组 [2, 5, 3, 1, 4] 和右子数组 [7, 8] 重复上述过程。

直到所有子数组只剩下一个或零个元素，排序完成。


average time complexity：O(n log n)

the worst time complexity ：O(n²) （当数组已经有序或逆序，且基准选择不当时）
when the pivot point is the largest the one within the arary ,so there is no left subside 

space complexity ：O(log n)
